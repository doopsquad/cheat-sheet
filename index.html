HTML Element
a unit of content that is defined by its tags and text inside
<p>Hello world</p>
HTML Structure
When an element is nested inside another, the nested one is the child and the outer one is the parent
<div> <p>Hello world</p> </div>
Semantic HTML
Provide info about the element's use, optimizes SEO
<nav>, <header>, <article>, etc.
Body tags
content inside displayed to screen
<body>hello</body>
Head tags
metadata, not displayed on screen
Font links, links to stylesheets, viewport, etc.
Title tag
sets the title of the webpage
<title>The Cheat-Sheet<title>
Div tag
block container, holds html elements
look at HTML structure example
Span tags
inline container, used to hold small pieces of text
<span>joe</span>
Br tag
Used for line breaks, doens't need closing tag. Can be used inside other tags
<br><span>joe</span>
Ul tag
Makes an unordered list, individual list items have to be added using <li>
<ul> <li></li> <li></li> </ul>
Ol tag
Makes an ordered list.
Look to Ul for example
Video tag
Used to hyperlink videos to your file. A lot of attributes can be used.
<video src="" heigh="" width="" controls autoPlay loop></video>
Tables
<tr> is used to make rows. <td> cell data. <th> works much like td, but semantic. <tbody>, <thead>, <tfoot> are also semantic. All need closing tags.
